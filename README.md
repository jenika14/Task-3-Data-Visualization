# Data Visualization Project

## Objective
To represent data visually using charts and graphs for better understanding.

## Tools Used
* Python
* Pandas
* Matplotlib
* Seaborn

## Visualizations Created
* Histogram (data distribution)
* Bar chart (category comparison)
* Line plot (trend analysis)
* Heatmap (correlation)

## Insights
* Identified trends and patterns in dataset
* Compared different categories
* Found relationships between variables
